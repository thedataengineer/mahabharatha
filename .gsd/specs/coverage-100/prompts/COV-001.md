# COV-001: Create MockMergeCoordinator with full merge flow simulation

## Overview
Enhance existing MockMergeCoordinator in tests/mocks/mock_merge.py to support full merge flow simulation.

## Files to Modify
- `tests/mocks/mock_merge.py`

## Requirements

### Enhance MockMergeCoordinator

The MockMergeCoordinator should support:

1. **Configure method** for setting up simulation behavior:
```python
def configure(
    self,
    timeout: bool = False,
    fail_merge: bool = False,
    fail_gates: bool = False,
    conflict_files: list[str] | None = None,
    staging_branch: str = "staging/test",
) -> None:
    """Configure mock behavior for testing."""
    self._timeout = timeout
    self._fail_merge = fail_merge
    self._fail_gates = fail_gates
    self._conflict_files = conflict_files or []
    self._staging_branch = staging_branch
```

2. **prepare_merge** - returns configurable staging branch name
3. **execute_merge** - simulates merge with conflict handling
4. **run_pre_merge_gates** - returns configurable gate results
5. **full_merge_flow** - chains all operations with configurable outcomes

### Add tracking for assertions:
```python
self._prepare_calls: list[dict] = []
self._merge_calls: list[dict] = []
self._gate_calls: list[dict] = []
self._flow_calls: list[dict] = []
```

## Verification
